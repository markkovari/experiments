# petClinic

## development

### install

1. use [uv](https://docs.astral.sh/uv/) package manager

```console
uv sync
```

1. run the app

```console
uv run manage.py runserver
```

#### justfile

For convenience there was a justfile [./justfile](./justfile) added.
Please download [just](https://github.com/casey/just) to use it.
